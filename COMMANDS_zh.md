## 网络工具

所有网络命令需要 wasmtime 网络标志：

```bash
wasmtime -W exceptions=y \
  -S tcp=y -S udp=y -S inherit-network -S allow-ip-name-lookup=y \
  busybox.wasm wget http://example.com
```

| 命令 | 状态 | 说明 |
|------|------|------|
| `wget` | ⚠️ | 需要网络标志。没有 `allow-ip-name-lookup=y` 时 DNS 可能失败 |
| `nc` / `netcat` | ⚠️ | 需要网络标志。基本 TCP 连接可用 |
| `telnet` | ⚠️ | 需要网络标志。交互式 TTY 可能受限 |
| `tcpsvd` | ⚠️ | 需要网络标志 |
| `udpsvd` | ⚠️ | 需要网络标志 |
| `ftpget` | ⚠️ | 需要网络标志 |
| `ftpput` | ⚠️ | 需要网络标志 |
| `whois` | ⚠️ | 需要网络标志 |
| `nslookup` | ⚠️ | DNS stub 不完整；部分查询类型可能失败 |
